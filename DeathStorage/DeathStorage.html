Description

DeathStorage is a unique Minecraft plugin that revolutionizes the death experience! Instead of losing your items or having them scattered on the ground, this plugin creates or drops a special death chest that safely preserves all your inventory items in an organized chest GUI..

Perfect for survival, PvP, BOXPVP and any server wanting better death mechanics!

version:1.0.1 Server-Version:1.21 - 26.2 Dependencies:None Server-Software:Spigot/Paper
->

Key Features:

  • ➤ Safe death chests with all items preserved
  • ➤ Floating holograms with player info
  • ➤ One-click "Collect All" button
  • ➤ 7 customizable death sounds (or none)
  • ➤ Detailed Death Messages - Know exactly what killed you
  • ➤ Beautiful visual effects
  • ➤ Can control the despawn time on the config
->

Commands:

  • - /deathstorage help
  • - /deathstorage sound <sound>
  • - /deathstorage reload
  • - /deathstorage about
⚙

Chat Format Configuration Options:

config.yml 
# DeathStorage Configuration
# Available sounds: ENDERMAN_DEATH, WITHER_DEATH, LIGHTNING, DRAGON_GROWL, GHAST_SCREAM, TOTEM, NONE

# Death chest despawn time in seconds (0 = never despawn)
chest-despawn-time: 300

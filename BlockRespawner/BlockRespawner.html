BlockRespawner

active

BlockRespawner Set how long a block takes to respawn, choose exactly what it drops when broken, customize the sound played when it respawns, and much more.

->

Description

Tired of boring, static block respawn mechanics? BlockRespawner revolutionizes the way blocks regenerate, with custom sounds, particles, broadcasts, drops, and more!, Whether you're creating custom mines, resource generators, or unique gameplay mechanics, the possibilities are endless! Perfect for PvP arenas, RPG worlds, BOXPVP , custom loot systems, and minigames.

version:1.0 Server-Version:1.21 - 26.2 Dependencies:None Server-Software:Spigot/Paper
->

Key Features:

  • ➤ Custom Respawn Timers - Set delays in seconds
  • ➤ Multiple Respawners - Create unlimited unique respawn points
  • ➤ Location Sticks - Easily set respawn locations with a magic stick
  • ➤ Particle Effects - Flames, hearts, smoke, portals, and more
  • ➤ Particle Patterns - Circle, Tornado, or Default animations
  • ➤ Custom Sounds - Play sounds on respawn or block break
  • ➤ Global Announcements - Alert players when blocks respawn or are stolen
  • ➤ Placeholder Support - %player%, %block%, and %location%
  • ➤ Hex Color Codes - Create colorful messages and gradients
  • ➤ Override Default Drops - Diamonds, custom items, or anything you choose
  • ➤ Special Rewards - Perfect for loot crates and secret rewards
  • ➤ GUI-Based Setup - No confusing configs, just click and set
  • ➤ Permission Support - Includes blockrespawner.use
  • ➤ Lightweight & Optimized - No lag, even with 100+ respawners
->

Planned Features (Coming Soon!):

Commands:

  • - /br create <name> - Create a new respawner
  • - /br gui <name> - Open configuration GUI
  • - /br list - List all respawners
  • - /br delete <name> - Delete a respawner
